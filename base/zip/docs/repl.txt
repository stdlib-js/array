
{{alias}}( arrays )
    Zips one or more arrays.

    The function assumes that the list of arrays to be zipped all have the same
    length.

    Parameters
    ----------
    arrays: ArrayLikeObject<ArrayLikeObject>
        List of arrays to zip.

    Returns
    -------
    out: Array<Array>
        Output array.

    Examples
    --------
    > var x = [ 1, 2 ];
    > var y = [ 3, 4 ];
    > var out = {{alias}}( [ x, y ] )
    [ [ 1, 3 ], [ 2, 4 ] ]
