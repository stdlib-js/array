
{{alias}}( x, value, start, end )
    Returns a new array with all elements within a specified range replaced
    with a provided value.

    Negative indices are resolved relative to the last array element, with the
    last element corresponding to `-1`.

    Parameters
    ----------
    x: ArrayLikeObject
        Input array.

    value: any
        Fill value.

    start: integer
        Starting index (inclusive).

    end: integer
        Ending index (exclusive).

    Returns
    -------
    out: ArrayLikeObject
        Output array.

    Examples
    --------
    > var x = [ 1, 2, 3, 4 ];
    > {{alias}}( x, 5, 1, 3 )
    [ 1, 5, 5, 4 ]
    > {{alias}}( x, 6, -3, -1 )
    [ 1, 6, 6, 4 ]


{{alias}}.assign( x, value, start, end, out, stride, offset )
    Copies elements from one array to another array and replaces all elements
    within a specified range with a provided value.

    Negative indices are resolved relative to the last array element, with the
    last element corresponding to `-1`.

    Parameters
    ----------
    x: ArrayLikeObject
        Input array.

    value: any
        Fill value.

    start: integer
        Starting index (inclusive).

    end: integer
        Ending index (exclusive).

    out: ArrayLikeObject
        Output array.

    stride: integer
        Output array stride.

    offset: integer
        Output array offset.

    Returns
    -------
    out: ArrayLikeObject
        Output array.

    Examples
    --------
    > var x = [ 1, 2, 3, 4 ];
    > var out = [ 0, 0, 0, 0 ];
    > var arr = {{alias}}.assign( x, 5, 1, 3, out, 1, 0 )
    [ 1, 5, 5, 4 ]
    > var bool = ( arr === out )
    true
